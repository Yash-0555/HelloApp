# \# HelloApp

# 

# \## Overview

# HelloApp is a simple application to demonstrate Git/GitHub workflows, feature branches, and use case implementation.

# 

# \## Project Purpose

# \- Organize source code and documentation

# \- Practice Git branching and pull requests

# \- Implement simple application use cases

# 

# \## Project Structure

# \- src/ → source code

# \- bin/ → compiled files

# \- docs/ → documentation

# 

# \## How to Run

# 1\. Clone repository

# 2\. Compile code (e.g., javac src/HelloApp.java -d bin)

# 3\. Run application (e.g., java -cp bin HelloApp)

# 

# \## Workflow

# \- main → production

# \- dev → integration branch

##### \- feature/\* → feature branches

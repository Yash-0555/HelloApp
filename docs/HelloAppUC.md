\# HelloApp Use Cases



\## UC1 – Display Hello World

\- The application should display "Hello World" when run.



\## UC2 – Display User Name

\- The application should ask the user for their name and display:

&nbsp; Hello <username>



\## UC3 – Display Greeting Message

\- The application should allow different greeting messages, e.g., "Good Morning", "Good Evening".



\## UC4 – Exit Application

\- The application should provide a way to exit safely.



\## UC5 – Log Activity (Optional)

\- The application should log actions into a log file for reference.
